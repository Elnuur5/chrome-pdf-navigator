# PDF Sentence Navigator

Chrome-ის გაფართოება PDF დოკუმენტებში წინადადებებს შორის ნავიგაციისთვის.

## ფუნქციები

- მუშაობს Chrome Extension Manifest V3-ზე
- PDF ფაილების დამუშავება ხდება ლოკალურად ბრაუზერში
- PDF ტექსტის ამოღება PDF.js ბიბლიოთეკის გამოყენებით
- წინადადებებს შორის გადაადგილება კლავიატურის საშუალებით:
  - TAB — შემდეგ წინადადებაზე გადასვლა
  - SHIFT + TAB — წინა წინადადებაზე დაბრუნება
- აქტიური წინადადება ვიზუალურად არის მონიშნული
- არ გამოიყენება გარე სერვერები

---

# ინსტალაცია

1. გახსენით Google Chrome.

2. გადადით მისამართზე:

```
chrome://extensions/
```

3. ჩართეთ:

```
Developer mode
```

(დეველოპერის რეჟიმი)

4. დააჭირეთ:

```
Load unpacked
```

(განტვირთული გაფართოების ჩატვირთვა)

5. აირჩიეთ პროექტის საქაღალდე:

```
chrome-pdf-navigator
```

6. გაფართოება გამოჩნდება Chrome-ის გაფართოებების სიაში.

---

# გამოყენების ინსტრუქცია

1. გახსენით გაფართოების PDF Viewer.

2. აირჩიეთ სასურველი PDF ფაილი.

3. დაელოდეთ PDF დოკუმენტის სრულად ჩატვირთვას.

4. გამოიყენეთ კლავიატურა:

```
TAB
```

შემდეგ წინადადებაზე გადასასვლელად.

გამოიყენეთ:

```
SHIFT + TAB
```

წინა წინადადებაზე დასაბრუნებლად.

აქტიური წინადადება ავტომატურად იქნება ვიზუალურად მონიშნული.

# კონფიდენციალურობა

PDF დოკუმენტების ყველა დამუშავება სრულდება ლოკალურად, მომხმარებლის ბრაუზერში.

არცერთი დოკუმენტის მონაცემი არ იგზავნება გარე სერვერებზე.
